# homebrew-apihop

Homebrew tap for [apihop](https://github.com/saturn4er/apihop) — a fast, open-source API client for REST, GraphQL, and WebSocket.

## Install

```bash
brew tap saturn4er/apihop
brew install apihop
```

## Usage

```bash
# Start the server (serves on http://localhost:3000)
apihop
```
